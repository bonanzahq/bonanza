## What was done

### Copilot PR review
Copilot left 3 comments on PR #49. Reviewed each:

1. **COPY --chown instead of RUN chown** -- Valid. Applied the fix.
   Build time for that layer dropped from 107s to 1.2s. This was already
   noted in the b4 plan as a future optimization but turned out to be
   trivial to fix now.
2. **SMTP_USERNAME/SMTP_PASSWORD fetch raises if unset** -- Wrong.
   Code already had `ENV.fetch("SMTP_USERNAME", nil)`. However, reviewing
   this caught a real bug: `SMTP_HOST` and `APP_HOST` used `ENV.fetch`
   without defaults, which broke `rails assets:precompile` during
   `docker build` (no env vars in build context). Added defaults.
3. **entrypoint.sh not executable** -- Wrong. Already `100755` in git.
